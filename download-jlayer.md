# Descargar JLayer para soporte MP3

## Opción 1: Descarga directa (Recomendada)

1. Ve a: https://www.javazoom.net/javalayer/javalayer.html
2. O descarga directamente desde Maven Central: https://repo1.maven.org/maven2/javazoom/jlayer/1.0.1/jlayer-1.0.1.jar
3. Guarda el archivo `jlayer-1.0.1.jar` en la carpeta `lib` del proyecto

## Opción 2: Usar wget/curl (si tienes instalado)

```bash
cd lib
curl -O https://repo1.maven.org/maven2/javazoom/jlayer/1.0.1/jlayer-1.0.1.jar
```

## Verificar

Después de descargar, deberías tener:
- `lib/jlayer-1.0.1.jar`

Luego reinicia NetBeans para que reconozca la nueva librería.
